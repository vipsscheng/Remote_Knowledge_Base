# AI Agent Core Skill

## 技能描述
AI代理核心技能，支持自我进化、自动化工具和知识库管理。

## 功能特性
- **自我进化系统**：自动自我修改、自我完善、自我进化和自我生长
- **自动化工具**：代码生成、测试、部署、工作流自动化等
- **知识库管理**：结构化知识存储和版本控制
- **多线程支持**：后台运行自我进化系统
- **优先级任务处理**：按权重执行任务
- **AI模型检测**：自动检测和选择可用的AI模型
- **Trae版本兼容**：同时支持Trae国际版和国内版
- **多模型支持**：支持国际模型（GPT-4、Claude、Gemini）和国内模型（文心一言、讯飞星火、通义千问、豆包）

## 安装说明
1. 运行 `python md_installer.py` 安装技能
2. 技能会自动检测Trae版本并创建相应的技能目录
3. 技能会复制到用户目录下的 `.trae/skills` 和 `.trae-cn/skills` 目录中
4. 运行 `start_skill.bat` 启动技能

## 使用方法
- 技能启动后会在后台运行自我进化系统
- 系统会自动检测可用的AI模型并使用
- 技能会定期分析系统状态并进行自我改进

## 配置文件
- `config.json`：技能配置文件
- `automation/evolution/system_config.json`：自我进化系统配置

## 日志
- 自我进化系统日志存储在 `技能文件夹/自动化工具/evolution/logs` 目录中
